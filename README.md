## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Chithra185/MY_PORTFOLIO.git
```

Move to project folder:

```bash
cd MY_PORTFOLIO
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:5173
```
